# AIDeer-Unity3D
Unity3D Project for AI-Deer

Unity3D Project that includes GameObject "animals".  C# sharp scripts for simple animal behaviors, c# scripts to 
implement zermoMQ communications with Opencog, DLLs for specialized "NetMQ for Unity", c# scripts to implement
"what GameObjects can I see?"  including accounting for occlusion, c# script to implement a prototype of "AI Deer" 
that sends description of scene to Opencog and receives back action instrunctions.
